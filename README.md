# Recipe Sharing System

This project is designed to streamline the process of sharing and discovering recipes online, replacing traditional methods like handwritten notes or scattered websites. Contributors can add new recipes, including ingredients, instructions, and images. Users can browse available recipes, search by category or ingredients, and request to save or try them. Once a recipe is submitted, an admin will review it and either approve or reject it. If approved, the recipe becomes publicly visible, and the contributor may receive a notification; if rejected, they will be notified with feedback.

## Usage

## Step 1: Install Dependencies

Navigate to both the backend and frontend directories and install the required dependencies by running:

npm install

## Step 2: Start the Backend

Go to backend directory and start the server using:

npm start

## Step 3: Start the Frontend

Go to frontend directory and start the frontend server using:

npm run dev

## Step 4:

After starting the server, if you want to log in as an admin, you can find the admin credentials in the app.js file in the backend. If you want to sign up as a user, use a real Gmail account, as users will receive email notifications about recipes approvals.
